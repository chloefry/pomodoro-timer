# Pomodoro Timer CLI Tool
A timer technique that improves focus by breaking work into 25 minute segments with 5 minute breaks between. After 4 full cycles (1 hour of work), a 15 minute break is required.

## Pomodoro Functionality 
The timer will be run through the command line and sound an alarm that indicates when to begin working and a seperate sound for breaks. The user can exit this cycle at any time. The timer will be run through the command line and written in rust.
